    jumeg.connectivity.plot_grouped_connectivity_circle¶

    jumeg.connectivity.plot_grouped_connectivity_circle(yaml_fname, con, orig_labels, replacer_dict, labels_mode=None, node_order_size=68, indices=None, out_fname='circle.png', title=None, subplot=111, include_legend=False, n_lines=None, fig=None, show=True, vmin=None, vmax=None, colormap='hot', colorbar=False, colorbar_pos=(-0.25, 0.05), symmetric_cbar=False, bbox_inches=None, tight_layout=None, **kwargs)¶

    Plot the connectivity circle grouped and ordered according to groups in the yaml input file provided.

    orig_labelslist of str

    Label names in the order as appears in con.

    labels_modestr | None

    ‘blank’ mode plots no labels on the circle plot, ‘cortex_only’ plots only the name of the cortex on one representative node and None plots all of the orig_label names provided.

    bbox_inches : None | ‘tight’

    tight_layout : bool

    NOTE: yaml order fix helps preserves the order of entries in the yaml file.
